# GeneSys | Self-Healing Cyber-Resilient System

A biological-inspired cyber defense system prototype built with Next.js and Genkit.

## Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Frontend**: [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) & [ShadCN UI](https://ui.shadcn.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/) (Scroll-linked sequences & Real-time diffs)
- **AI Framework**: [Genkit](https://firebase.google.com/docs/genkit)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## Key Features

### 1. Cinematic DNA Sequence
A high-performance HTML5 Canvas animation synced to user scroll, representing the restoration of a compromised system's "Genetic Code."

### 2. Live Repository Engine
An interactive, Git-aware codebase viewer that demonstrates real-time "self-healing" logic. It visualizes:
- **Detection**: Identifying threat vectors (SQLi, DDoS, Zero-Day).
- **Response**: Generating Git-style diff patches (Additions/Removals).
- **Learning**: Committing reinforced antibodies to "Genetic Memory."

### 3. Mutation Simulator
A stress-test interface for triggering diverse cyber-threats, each with unique visual artifacts (glitch effects, traffic spikes, encryption freezes) and targeted logical responses.

## Getting Started

To explore the core logic, start with:
- `src/app/page.tsx`: The cinematic intro sequence.
- `src/app/core/page.tsx`: The System Core control interface.
- `src/lib/genesys-engine.ts`: The reactive state machine driving the simulation.
